# Regression Model Evaluation

## Dataset
- California Housing Dataset

## Models Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Gradient Boosting Regressor
- Support Vector Regressor (SVR)

## Performance Metrics
- Mean Squared Error (MSE)
- Mean Absolute Error (MAE)
- R² Score

## Best Model
- **Random Forest Regressor**
  - **MSE**: 0.2552
  - **MAE**: 0.3274
  - **R²**: 0.8053

## Worst Model
- **Linear Regression**
  - **MSE**: 0.5559
  - **MAE**: 0.5332
  - **R²**: 0.5758
